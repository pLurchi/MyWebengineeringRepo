Tabelle mit HTML Elementen.

Element Bedeutung Inline-/ Blockelement Semantisches Element
<a> Verweis (Hyperlink) und Anker für Verweise Inlineelement Semantisches Element
<abbr> Abkürzung Inlineelement Semantisches Element
<acronym> Akronym Inlineelement
<address> Kontaktangabe zum Autor eines Inhalts (Credits) Blockelement
<area> Verweis-sensitiver Bereich in einer Image-Map-Grafik
<b> Stilistische Hervorhebung im Text (üblicherweise fett dargestellt) Inlineelement Semantisches Element
<base> Basis-URL für Referenzen
<bdo> Richtungsänderung bei bidirektionalem Text Inlineelement Semantisches Element
<big> Großschrift Inlineelement
<blockquote> Zitat in Form eines eigenen Textabsatzes Blockelement
<body> Inhaltsbereich eines HTML-Dokuments
<br> Erzwungener Zeilenumbruch an der betreffenden Stelle Inlineelement Semantisches Element
<button> Schaltfläche in einem Formular Inlineelement
<caption> Tabellenüberschrift / -unterschrift
<cite> Zitat innerhalb eines Absatzes Inlineelement Semantisches Element
<code> Quelltext-Auszug innerhalb eines Absatzes Inlineelement Semantisches Element
<col> Tabellenspalte
<colgroup> Gruppe von Tabellenspalten
<dd> Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element
<del> gelöschter Inhalt (Änderungsmarkierung) Inline-/Blockelement
<dfn> Definition innerhalb eines Absatzes Inlineelement Semantisches Element
<div> Allgemeiner Bereich Blockelement
<dl> Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw. Blockelement
<dt> Zu definierender Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw.
<em> emphatischer Ausdruck innerhalb eines Absatzes (z.B. überspitzte Formulierung) Inlineelement Semantisches Element
<fieldset> Gruppe von Elementen in einem Formular, Reiter in einem Dialog Blockelement
<form> Formular, Dialog Blockelement
<h1> - <h6> Überschrift 1. bis 6. Ordnung Blockelement
<head> Kopfbereich eines HTML-Dokuments
<hr> Trennlinie
<html> Dokumentelement von HTML-Dokumenten, schließt das gesamte Dokument ein
<i> Kursivschrift Inlineelement Semantisches Element
<img> Referenzierte und eingebettete Grafik Inlineelement
<input> Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw.
<ins> Eingefügter Inhalt (Änderungsmarkierung) Inline-/Blockelement
<kbd> Darstellung eines über Tastatur einzugebenden Textes im Text Inlineelement Semantisches Element
<label> Beschriftungstext eines Formularfeldes Inlineelement
<legend> Überschrift für eine fieldset-Gruppe von Elementen in einem Formular
<li> Listenpunkt in einer ul-, ol- oder menu-Liste
<link> Logische Beziehungen der aktuellen Webseite zu anderen Webseiten
<map> Bereich für verweis-sensitive Grafiken Inlineelement
<meta> Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver
<noscript> Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist Blockelement
<object> Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw. Inlineelement
<ol> Geordnete (z.B. nummerierte) Liste Blockelement
<optgroup> Gruppe von Auswahlmöglichkeiten in einer Auswahlliste
<option> Auswahlmöglichkeit in einer Auswahlliste
<p> Textabsatz Blockelement
<param> Parameter eines Objekts bei object-Element
<pre> Präformatierter Text (alle Whitespace-Zeichen werden im Browser so dargestellt wie im Text enthalten) Blockelement
<q> Wörtliche Rede in einem Text Inlineelement Semantisches Element
<samp> Beispieltext in einem dokumentierenden Text Inlineelement Semantisches Element
<script> Scriptbereich für JavaScript und andere Scriptsprachen Inlineelement
<select> Auswahlliste in einem Formular Inlineelement
<small> kleiner dargestellter Text als Normalschrift Inlineelement Semantisches Element
<span> allgemeiner Bereich innerhalb eines Absatzes Inlineelement Semantisches Element
<strong> stark betonter Inhalt im Text Inlineelement Semantisches Element
<style> Bereich für Stylesheet-Definitionen, z.B. in CSS
<sub> Tiefgestellter Text Inlineelement Semantisches Element
<sup> Hochgestellter Text Inlineelement Semantisches Element
<table> Tabelle Blockelement
<tbody> Tabellenkörper mit den eigentlichen Daten
<td> Tabellenzelle
<textarea> Mehrzeiliger Eingabebereich in einem Formular Inlineelement
<tfoot> Fußbereich in einer Tabelle, z.B. für summarische Daten
<th> Tabellenkopfzelle
<thead> Kopfbereich einer Tabelle
<title> Titel eines HTML-Dokuments
<tr> Tabellenzeile
<tt> <<Fernschreiber>>-Text Inlineelement
<ul> Unsortierte Liste (Aufzählungsliste) Blockelement
<var> Variabler Name in einem dokumentierenden Text Inlineelement Semantisches Element
